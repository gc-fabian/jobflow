# Seguridad

Este repositorio no debe contener tokens, contraseñas, CVs finales ni postulaciones reales con datos personales.

Si un token fue pegado en chat, logs o commits:
1. Revocarlo inmediatamente en https://github.com/settings/tokens
2. Crear uno nuevo solo para el uso puntual.
3. Preferir `gh auth login` o un prompt interactivo antes que escribir tokens en archivos.

Archivos locales ignorados:
- `data/jobs.json`
- `exports/`
- `dashboard.log`
- `.env`
- CVs PDF y paquetes de postulación.
